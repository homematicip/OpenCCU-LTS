OpenCCU-LTS (**L**ong **T**erm **S**ervice) ist ein kostenloses, alternatives Betriebssystem für die Smart Home Zentrale CCU3 von eQ-3. Die Long-Term-Service Version ist ein stabiler und zuverlässiger Entwicklungszweig des communitygetriebenen [OpenCCU](https://github.com/OpenCCU/OpenCCU) Projekts. Die Entwicklung wird von eQ-3 unterstützt, ein vollständiger Support kann jedoch nicht gewährleistet werden.


## Systemvoraussetzungen

OpenCCU-LTS ist ausschließlich für den Betrieb auf einer [CCU3](https://homematic-ip.com/en/product/smart-home-ccu3-central-control-unit) oder einem [ELV Charly](https://de.elv.com/elvwissen/elvprojekte/zentralen-grundfunktionen/charly-die-naechste-generation-der-smart-home-zentrale) vorgesehen. 

Für den Einsatz auf anderer Hardware verwenden Sie bitte die [OpenCCU](https://github.com/OpenCCU/OpenCCU).

## Installation
1. Download:<br>Bitte laden Sie das aktuelle Release-Archiv (OpenCCU-X.XX.XX.YYYYMMDD-ccu3.tgz) herunter. Die Release-Archive sind unter [Releases](https://github.com/homematicip/OpenCCU-LTS/releases) zu finden.
2. Installation:<br>
3. WebUI aufrufen<br>


### 2 Installation
Installieren Sie das Archiv über die WebUI unter Einstellungen -> Systemsteuerung -> Zentralen-Wartung
### 3 WebUI aufrufen
Rufen Sie die IP des Geräts im Browser auf, um die OpenCCU-LTS-WebUI zu öffnen.
